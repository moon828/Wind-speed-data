# Wind-speed-data
Steady wind, gusty wind, and wind shear data（10kHz Sampling rate）
**稳态风参数**

1.  输出强度范围：20%、30%、40%、50%、60%、70%、80%、90%、99%；
2.  稳定持续时间：每个工况下测试60s；

**动态风**

动态风分为**正弦波风、方波风、三角波风**三类，每类均包含 "固定频率" 和 "固定幅值" 两种控制策略，且三类风况参数完全一致，
以下分表呈现（测量2个周期），数据文件命名格式为"上限-下限-周期"，例如"55-45-40即为风墙风扇PWM上限占空比为55%，下限为45%，周期为40秒

| 控制策略 | 基准占空比 D₀ | 波动幅值 ΔD | 频率 f (Hz) |
| :--- | :--- | :--- | :--- |
| 固定频率 (f=0.025Hz) | 50% | 2% | 0.025 |
| 固定频率 (f=0.025Hz) | 50% | 5% | 0.025 |
| 固定频率 (f=0.025Hz) | 50% | 10% | 0.025 |
| 固定频率 (f=0.025Hz) | 50% | 20% | 0.025 |
| 固定幅值 (ΔD=5%) | 50% | 5% | 0.05 |
| 固定幅值 (ΔD=5%) | 50% | 5% | 0.025 |
| 固定幅值 (ΔD=5%) | 50% | 5% | 0.0125 |
| 固定幅值 (ΔD=5%) | 50% | 5% | 0.00625 |

**风切变**

风切变横向、纵向风切变各设置 10 个空间关键点用于数据采集；采集60s

| 工况名称 | 切变方向 | 切变间隔 | 切变范围|
| :--- | :--- | :--- | :--- |
| 横向- 1% 间隔 - 升切变 | 横向 | 1% | 30%-70% |
| 横向 - 2% 间隔 - 升切变 | 横向 | 2% | 20%-100% |
| 纵向 - 1% 间隔 - 升切变 | 纵向 | 1% | 30%-70% |
| 纵向 - 2% 间隔 - 升切变 | 纵向 | 2% | 20%-100% |
